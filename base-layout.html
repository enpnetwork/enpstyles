Clearfix

.clearfix {
  overflow: hidden;
  display: inline-block;
}

.clearfix {
  display: block;
}

.clearfix:after {
  visibility: hidden;
  display: block;
  font-size: 0;
  content: " ";
  clear: both;
  height: 0;
}

* html .clearfix {
  zoom: 1;
}

*:first-child + html .clearfix {
  zoom: 1;
}

Pie Clearfix

.pie-clearfix {
  *zoom: 1;
}

.pie-clearfix:before, .pie-clearfix:after {
  display: table;
  content: "";
}

.pie-clearfix:after {
  clear: both;
}

.float-left, .float-right

.float-left floats an object left, .float-right floats an object right. 

.float-left {
  display: inline;
  float: left;
}

.float-right {
  display: inline;
  float: right;
}

.hidden

Hides the element. You can't see me. 

.hidden {
  display: none;
}

.text-align-[left, center, right]

Aligns text. 

.text-align-left {
  text-align: left;
}

.text-align-center {
  text-align: center;
}

.text-align-right {
  text-align: right;
}

.nowrap

Prevents whitespace wrapping. 

.nowrap {
  white-space: nowrap;
}
